elements

@property NSMutableDictionary *elements
Return value

A dictionary of all of the NSBitmapImageReps that will be used for the ICO file. Keys would either be an NSNumber representation of the element used or a size from NSStringFromSize

Discussion

All of the representations go in here, unless they are null.
